Python Fundamentos

Referência explicada para scripts e análise de dados.

if / else

Condicionais

Explicação

O bloco if serve para fazer o código "tomar decisões". Ele avalia se uma condição é verdadeira. Se não for, ele tenta o elif (else if). Se nenhuma funcionar, ele executa o else como última opção.
Atenção: A indentação (espaço antes da linha) é obrigatória no Python!

Exemplo 
idade = 18
if idade >= 18:
    print("Maior") # Executa se for verdade
elif idade >= 16:
    print("Vota")  # Executa se o primeiro if falhar
else:
    print("Menor") # Se nada for verdade
for

Repetições

Explicação

Use o for quando quiser repetir uma ação várias vezes ou percorrer uma lista inteira item por item.

Exemplo: Range

Contador numérico. O último nº não conta. 

for i in range(5):
    print(i) # 0, 1, 2, 3, 4
Exemplo: Lista

Itera item por item. 

for item in lista:
    print(item)
[ ]

Listas

Estrutura Básica

Explicação

São sequências ordenadas e mutáveis. Use colchetes []. O índice do primeiro item é sempre 0.

Exemplo 
frutas = ["Maçã", "Banana", "Uva"]

# Acessando (Lendo)
print(frutas[0])  # "Maçã"
print(frutas[-1]) # "Uva" (último)

Manipulação

Explicação

Como adicionar, remover e buscar itens dentro da lista.

Exemplo 
lista = ["A", "B", "C"]

# ADICIONAR
lista.append("D")      # No final: ["A", "B", "C", "D"]
lista.insert(0, "Z")   # No início (índice 0): ["Z", "A"...]

# REMOVER
lista.remove("B")      # Remove pelo VALOR (o primeiro que achar)
item = lista.pop(2)    # Remove pelo ÍNDICE e retorna o valor

# BUSCAR / VERIFICAR
if "A" in lista:
    print("A letra A está na lista!")

# TAMANHO
print(len(lista))      # Conta quantos itens tem

Fatiamento (Slicing)

Explicação

Recurso poderoso para pegar "pedaços" da lista. A sintaxe é [início : fim].
Regra de ouro: O índice do final NÃO é incluído no resultado.

Exemplo 
nums = [0, 10, 20, 30, 40]

# Do índice 1 até o 3 (o 3 fica de fora)
print(nums[1:3]) # Saída: [10, 20]

# Do começo até o 2
print(nums[:2])    # Saída: [0, 10]

# Do 2 até o final
print(nums[2:])    # Saída: [20, 30, 40]

Chave e Valor. Use chaves {}.

Manipulação

Explicação

Adicionar, remover chaves e buscar valores de forma segura.

Exemplo 
aluno = {"nome": "Leo"}

# ADICIONAR / ALTERAR
aluno["curso"] = "Python"  # Cria nova chave "curso"
aluno["nome"] = "Leonardo" # Se já existe, atualiza o valor

# REMOVER
del aluno["curso"]       # Remove a chave "curso" pra sempre
nota = aluno.pop("nota", None) # Remove e retorna valor (sem erro se não existir)

# BUSCAR SEGURA (Evita erro se chave não existe)
print(aluno.get("idade", "Não informado")) 

# VERIFICAR CHAVE
if "nome" in aluno:
    print("Existe a chave nome")

Pandas (Análise de Dados)

O Pandas é a biblioteca padrão para manipular tabelas (DataFrames). Ele é infinitamente mais rápido que o Excel para grandes volumes de dados.

pd.read_csv

Leitura de Arquivos

Explicação

Antes de analisar, precisamos carregar o arquivo e entender a estrutura dos dados (quais colunas existem, se tem valores vazios, etc).

Exemplo 
import pandas as pd

# Lendo o arquivo CSV para a memória (Assumindo que o arquivo existe)
df = pd.read_csv('vendas.csv')

# Visualização inicial
df.head()       # Mostra as 5 primeiras linhas (padrão)
df.head(10)     # Mostra as 10 primeiras linhas
df.info()       # Mostra resumo técnico (tipos de dados e contagem de não-nulos)
df.describe()   # Gera estatísticas automáticas (Média, Mínimo, Máximo)
df[ ]

Seleção e Filtros

Explicação

Raramente usamos a tabela inteira. Na maioria das vezes, queremos filtrar linhas específicas (ex: apenas vendas caras) ou selecionar colunas de interesse.

Exemplo 
import pandas as pd

# Criando dados de exemplo para funcionar
data = {
    'produto': ['Mouse', 'Teclado', 'Monitor', 'Laptop'],
    'preco': [50, 80, 450, 2500],
    'categoria': ['Acessórios', 'Acessórios', 'Periféricos', 'Eletrônicos']
}

# pd.DataFrame(): Converte o dicionário acima em uma Tabela estruturada
df = pd.DataFrame(data)

# Selecionar apenas uma coluna (retorna uma Série de dados)
precos = df['preco']

# Selecionar várias colunas (retorna um novo DataFrame menor)
resumo = df[['produto', 'preco']]

# FILTRAR LINHAS: Apenas onde o preço for maior que 100
vendas_caras = df[df['preco'] > 100]

# FILTRO DUPLO (AND): Acima de 100 E da categoria Eletrônicos
filtro = (df['preco'] > 100) & (df['categoria'] == 'Eletrônicos')
resultado = df[filtro]
df.dropna

Limpeza de Dados

Explicação

Dados do mundo real vêm sujos (com buracos/NaN). O Pandas facilita a limpeza e também a criação de novas colunas baseadas em cálculos.

Exemplo 
import pandas as pd

# Criando dados com valores nulos (None) para teste
data = {
    'produto': ['Mouse', 'Teclado', None],
    'preco': [50, None, 450]
}

# Transforma o dicionário em Tabela (DataFrame)
df = pd.DataFrame(data)

# Opção A: Apagar linhas que tenham QUALQUER valor vazio (NaN)
df_limpo = df.dropna()

# Opção B: Preencher os buracos vazios com zero (ou média)
df_preenchido = df.fillna(0)

# Criar uma NOVA coluna calculada (ex: dobrar o preço)
# Nota: Com fillna(0), o cálculo funcionará sem erros
df_preenchido['preco_dobro'] = df_preenchido['preco'] * 2

sns.scatterplot

Gráficos Estatísticos

Explicação

Construído em cima do Matplotlib. Ele entende o DataFrame do Pandas diretamente e cria gráficos complexos e bonitos com apenas 1 linha.

Exemplo 
import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

# Criando dados de exemplo
data = {
    'idade': [25, 30, 35, 40, 45],
    'salario': [3000, 4500, 5500, 7000, 8500],
    'genero': ['F', 'M', 'F', 'M', 'F'],
    'cargo': ['Jr', 'Pleno', 'Pleno', 'Senior', 'Senior']
}

# pd.DataFrame(): Converte o dicionário em um DataFrame (Tabela)
df = pd.DataFrame(data)

# Dispersão (Scatterplot) - Ótimo para ver correlação (ex: idade x salário)
sns.scatterplot(data=df, x='idade', y='salario', hue='genero')
# hue='genero': Pinta as bolinhas automaticamente baseado no gênero!
plt.show()

# Histograma - Para ver a distribuição dos dados (onde se concentram)
sns.histplot(data=df, x='idade', kde=True)
# kde=True: Desenha a linha de curva suave por cima das barras
plt.show()

# Boxplot - Essencial para achar Outliers (valores absurdos/erros)
sns.boxplot(data=df, x='cargo', y='salario')
plt.show()
Seaborn + Matplotlib

Integração

Explicação

O Seaborn é construído sobre o Matplotlib. Isso significa que eles não são concorrentes, mas parceiros! Você usa o Seaborn para criar o gráfico complexo facilmente e o Matplotlib para ajustar os detalhes finais (títulos, eixos, tamanho).

Exemplo
Exemplo de Trabalho em Equipe 
import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

# Criando dados de exemplo
data = {
    'idade': [25, 30, 35, 40, 45],
    'salario': [3000, 4500, 5500, 7000, 8500],
    'genero': ['F', 'M', 'F', 'M', 'F']
}

# Converte o dicionário em DataFrame
df = pd.DataFrame(data)

# 1. SEABORN: Cria o gráfico bonito e estatístico
sns.scatterplot(data=df, x='idade', y='salario', hue='genero')

# 2. MATPLOTLIB: Personaliza os detalhes
plt.title("Relação Idade vs Salário por Gênero") # Título
plt.xlabel("Idade do Funcionário")               # Rótulo X
plt.ylabel("Salário Anual ($)")                  # Rótulo Y
plt.grid(True, linestyle='--')                   # Grade tracejada
plt.show()
Home Git

Comandos do Git

Entenda o que cada comando faz no seu computador (localmente).

git init

Criando do Zero

Explicação

Este comando transforma uma pasta comum do seu computador em um repositório Git. Ao rodar isso, o Git cria uma pasta oculta chamada .git onde ele vai guardar todo o histórico de alterações.

Exemplo
# 1. Entre na pasta do seu projeto
cd meus-projetos/meu-site
# 2. Inicie o Git
git init
Initialized empty Git repository in .../.git/
git add

Preparando (Staging)

Explicação

O Git não salva automaticamente quando você edita um arquivo. Você precisa avisar quais arquivos quer incluir no próximo "save". O add coloca o arquivo na Área de Preparação (Staging Area). É como colocar produtos no carrinho de compras antes de pagar.

Exemplo
# Adicionar um arquivo específico
git add index.html
# OU adicionar TODOS os arquivos modificados (mais comum)
git add .
git commit

Salvando (Histórico)

Explicação

Este é o comando que efetivamente salva uma versão no histórico. Ele pega tudo que estava no "carrinho" (Stage) e cria um pacote fechado com uma mensagem sua.

Exemplo
# A flag "-m" serve para escrever a mensagem direto
git commit -m "Cria a página inicial do site"
[master (root-commit) 3d1f4a] Cria a página inicial do site
1 file changed, 10 insertions(+)
git push

Enviando (Upload)

Explicação

Até agora, tudo aconteceu apenas no seu computador. Se o seu PC quebrar, você perde tudo. O push pega os seus commits locais e envia para um servidor remoto (como GitHub, GitLab ou um servidor da empresa).

Exemplo
# "origin" é o apelido do servidor remoto
# "main" (ou master) é o nome da branch principal
git push origin main
git pull

Atualizando (Download)

Explicação

Se você trabalha em equipe (ou em computadores diferentes), o código na nuvem pode mudar. O pull baixa as atualizações do servidor e as mistura (merge) com o seu código local automaticamente.

Exemplo
# Sempre faça um pull antes de começar a trabalhar!
git pull origin main
